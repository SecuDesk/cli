## `secudesk help [COMMANDS]`

Display help for secudesk.

```
USAGE
  $ secudesk help [COMMANDS] [-n]

ARGUMENTS
  COMMANDS  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for secudesk.
```
